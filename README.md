## 🛠️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/rahul11f/C-Journey.git
   cd C-Journey
   ```
2. Install dependencies (if applicable):
   ```bash
   npm install
   ```
3. Run the project locally:
   ```bash
   npm start
   # or npm run dev
   ```
